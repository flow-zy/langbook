# 安装指南

## 前提条件

在开始安装前，请确保您已经安装了以下软件：

- Node.js (>=18.16.0)
- pnpm (>=8.0.0)

## 安装步骤

1. **创建新项目**

```bash
pnpm create vuepress-theme-hope my-docs
cd my-docs
```

2. **安装依赖**

```bash
pnpm install
```

3. **启动开发服务器**

```bash
pnpm docs:dev
```

## 下一步

安装完成后，您可以继续阅读 [配置指南](configuration.md) 来了解如何配置您的网站。
